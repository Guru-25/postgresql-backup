# postgresql-backup
Backup PostgreSQL Database

## What it does?
Upload database dump to your Dropbox

## Action secrets

Assign secrets in Settings -> Secrets and variables -> Actions -> New repository secret

| Name                    |
| ----------------------- |
| `HOST`                  |
| `PORT`                  |
| `USERNAME`              |
| `PASSWORD`              |
| `DATABASE_NAME`         |
| `ZIP_PASSWORD`          |
| `DROPBOX_APP_KEY`       | 
| `DROPBOX_APP_SECRET`    |
| `DROPBOX_REFRESH_TOKEN` |
